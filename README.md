# scratch-next

## -- Instalation
```bash
# Clone scratch-next monorepo
git clone https://github.com/pitbulek0/scratch-next.git

# Install all deps (pnpm required!)
cd ./scratch-next
pnpm install

```

## -- Development
```bash
# Build
pnpm build
pnpm start

# Dev
pnpm dev

# Lint
pnpm lint:[package-name]

# Misc
pnpm commit # commitlint prompt
```
